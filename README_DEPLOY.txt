FORTNITE SPRITE TRACKER - DEPLOY

Best free path: GitHub Pages
1. Create a free GitHub account if needed.
2. Create a new repository named fortnite-sprite-tracker.
3. Upload all files from this folder: index.html, manifest.webmanifest, icon.svg, sw.js.
4. In GitHub repo: Settings > Pages.
5. Under Build and deployment, choose: Deploy from a branch.
6. Branch: main. Folder: /root.
7. Save. GitHub will give you a public URL.
8. Open that URL on iPhone Safari > Share > Add to Home Screen.

Alternative: Netlify Drop
1. Go to Netlify Drop in a browser.
2. Drag this whole folder in.
3. Netlify gives you an instant URL.

Notes:
- Progress saves locally on each device/browser.
- Use Export Backup / Import Backup to move progress between devices.
- Sprite list source: Fortnite.GG sprites page showed 61 released sprites on June 26, 2026.
